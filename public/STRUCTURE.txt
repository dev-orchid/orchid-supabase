COMMON WORKFLOWS
================

Update Page Styles:
1. Open css/pages/[page-name].css
2. Add styles at bottom of file
3. Save and refresh browser

Add Local Image:
1. Add image to images/[page-name]/
2. Update src/lib/images.ts
3. Verify on website

Customize Component:
1. Open css/components/[component].css
2. Modify or add classes
3. Apply in component files
